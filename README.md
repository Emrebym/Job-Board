# 💼 Job Board

A full-stack job listing platform where users can browse, search, and filter job postings by category, location, and type.

## Features

- **Job listings** — browse available positions with detailed descriptions
- **Search & filter** — find jobs by keyword, location, or category
- **Responsive layout** — optimized for all screen sizes
- **Clean interface** — easy navigation and readable job cards
